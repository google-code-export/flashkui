Packagecn.flashk.controls
Classpublic class Window
InheritanceWindow Panel UIComponent flash.display.Sprite
Subclasses Alert

Window 容器为其子项提供了标题栏、边框和内容区域。窗口默认允许用户拖动、更改大小和最大化/最小化。如果需要在窗口更改大小时实时排列窗口中的内容，请侦听resize事件。

通常，使用 Window 容器封装独立的应用程序模块。并弹出窗口显示。

例如，您可以在 Window 容器中包含一个背包窗口，你可以用代码关闭 Window 容器，或允许用户使用“关闭”按钮将其关闭。

useOwnSkin()method 
public function useOwnSkin(closeButton:InteractiveObject = null, dragWidthLess:Number = 0, dragHeight:Number = -1, dragPosX:Number = 0, dragPoxY:Number = 0):void

使用自定义皮肤，默认窗口拖动时会使用位图缓存，如果皮肤拖动时被裁剪，请将useDragCatch设定为false

Parameters

closeButton:InteractiveObject (default = null) — 对关闭按钮的引用，点击此按钮将关闭窗口，使用之后关闭将被从原来显示列表移除到窗口的子级（最上层）
 
dragWidthLess:Number (default = 0) — 要修改的拖动范围的值，默认是窗口宽度，如果是负值，则为窗口宽度减去多少
 
dragHeight:Number (default = -1) — 拖动的高度（直接设置值>=0）
 
dragPosX:Number (default = 0) — 拖动区域x位置
 
dragPoxY:Number (default = 0) — 拖动区域y位置
