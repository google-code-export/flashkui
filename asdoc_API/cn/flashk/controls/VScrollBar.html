Packagecn.flashk.controls
Classpublic class VScrollBar
InheritanceVScrollBar UIComponent flash.display.Sprite
Subclasses HScrollBar

Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0

VScrollBar 组件是一个竖直滚动条，可方便的设定DisplayObject滚动。

滚动条可设定缓冲（可以设定缓冲模式，大小值）/关闭缓冲。并且可以隐藏两端的小箭头。

滚动条可以设定当用户在滚动目标上按下鼠标时可以直接拖动内容
